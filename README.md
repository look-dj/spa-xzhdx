# 雪中悍刀行 后台项目

## 如何安装

```bash
# 安装依赖
$ npm install

# 在localhost:3000上启动项目
$ npm dev

# 打包项目用于部署到服务器上
$ npm build
$ npm start
# 打包并查看打包详细信息
$ npm run analyze
```
