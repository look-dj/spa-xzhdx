- 在使用 iconfont 的时候当使用的名称里面有中划线，或者名称里面有大写字母的时候需要在前面额外添加`iconfont`例子如下

```
当使用的图标名称是这种格式 => iconfont-hdsjkhdsjk-dskdj 的时候需要在前面加上iconfont
或者图标名称是这种格式 => iconfont-hdsjkhdsjkPskdj 的时候需要在前面加上iconfont
```
